# LEDcandle_v2

重构了程序，实现了包括根据柏林噪声生成动态火焰图案、贪吃蛇、雨滴、随机闪烁、温度显示、串口通信等效果。

硬件设计上精简了电路，实现单板完成所有功能；使用四层板，双面焊接原件，缩小了电路大小；添加了两个按键，通过短按、长按可以调整亮度、切换模式等。使用boot键实现模式内的功能键。

使用USB供电，可以插在充电器等。带一个串口转USB芯片（CH340N），方便插在电脑上。

发现linux也可以方便地实现USB串口通信，未来可能实现一下插在NAS上查看设备状态的功能。
