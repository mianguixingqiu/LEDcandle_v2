# LEDcandle_v2

重构了程序，实现了包括根据柏林噪声生成动态火焰图案、贪吃蛇、雨滴、代码雨、随机闪烁、温度显示、串口通信等效果。

硬件设计上精简了电路，实现单板完成所有功能；使用四层板，双面焊接原件，缩小了电路大小；优化了PCBlayout，原来的太丑陋了；添加了两个按键，通过短按、长按可以调整亮度、切换模式等。使用boot键实现模式内的功能键。

使用USB_A供电，可以插在充电器、电脑主板等。带一个串口转USB芯片（CH340N），方便插在电脑上。

原先的火焰抖动效果是存了好多图片循环播放，如今实现了柏林噪声算法生成，但由于硬件限制（TM1640无法控制单个像素的亮度），效果并不理想，抖动有些生涩。柏林噪声生成动态火焰图案参考了[星建工坊](https://github.com/HankWang0571/CandleLight)的算法，[他的B站视频](https://www.bilibili.com/video/BV1t5PjzQEy2)。他的仿真蜡烛灯效果更好，灯芯的结构设计也非常惊艳。TB店同名，建议玩了命的买。

使用html当上位机，实现将摄像头画面或选定视频文件通过串口发送到电子蜡烛上，实现酷炫的效果。我觉得这是本项目最棒的两点，有屏幕的地方就可以播放bad apple（视频会放的，请关注我的B站账号：[禁地萌希](https://space.bilibili.com/400728343)）。

发现linux也可以方便地实现USB串口通信，未来可能实现一下插在NAS上查看设备状态的功能。
